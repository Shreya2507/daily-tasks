//MODES
Normal Mode – for navigation and commands (default mode when you open a file) --> Press Esc to return here from any other mode.
Insert Mode – for typing text --> Press i to enter Insert Mode.
Visual Mode – for selecting text --> Press v (character-wise) or V (line-wise) from Normal Mode.

//CURSOR MOVEMENT
Arrow keys work OR “h/j/k/l”:
h → left
j → down
k → up
l → right

Word movement:
w → jump to start of next word
e → jump to end of word
b → jump to start of previous word

Line movement:
0 → beginning of line
^ → first non-blank character
$ → end of line


// OTHER COMMANDS
Insert text:
i → insert before cursor
a → append after cursor

Delete text:
x → delete character under cursor
dw → delete word
dd → delete entire line

Undo/Redo:
u → undo
Ctrl + r → redo

Copy/Paste (yank/put):
yy → copy (yank) line
p → paste after cursor
P → paste before cursor


//SAVE, QUIT
:q → quit (fails if there are unsaved changes)
:q! → quit without saving
:w → save (write)
:wq or ZZ → save and quit
